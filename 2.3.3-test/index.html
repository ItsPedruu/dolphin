About

Dolphin is a discord.js framework to easily create a discord bot. Dolphin has features such as Engines, Command Handler and Event Handler

• Bot Example • Documentation

Installing

npm install dolphin-discord

Basic Setup

main.js

const Dolphin = require('../main.js');

require('dotenv').config();

const config = require('./config.json');
const client = new Dolphin.Client(config);

client.login(process.env.token);

config.json

{
    "allowBots": false,
    "prefix": "!",
    "mentionAsPrefix": true,
    "commands": {
        "path": "./commands",
        "cooldown": 3
    },
    "useSendEmbed": true,
    "mainColor": "#FFFFFF",
}
