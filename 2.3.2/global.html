    Properties:
    Name Type Attributes Description
    allowBots boolean <nullable>
    		 Whether to allow bots to execute commands. Default: false
    prefix string <nullable>
    		 The default bot prefix for commands. Default: !
    mentionAsPrefix boolean <nullable>
    		 Whether to use the bot's mention as a prefix. Default: true
    owners array <nullable>
    		 An array of the bot's owners. This is normally used to set commands as owner only. Default: Empty Array
    commands CommandsRegisterOptions <nullable>
    		 Options to register the commands
    eventsPath string <nullable>
    		 The string path to the events folder
    disableReadyMessage boolean <nullable>
    		 Whether to display a console.log message saying the bot is online
    useSendEmbed boolean <nullable>
    		 Whether the text should appear in a nice embed when executing #Message.say() built-in function
    mainColor string <nullable>
    		 The main color used in the embeds for #Message.say() built-in function
    defaultCommand string <nullable>
    		 The default command that displays when the bot is mentioned
    enginesPath string <nullable>
    		 The string path to the engines folder
